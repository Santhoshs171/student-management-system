                                                              Student Management System
                                                              
Project Overview:
This project is a Student Management System built using React.js.
It allows users to add, edit, delete, search, and download student data using a simple and modern user interface.
All CRUD operations are handled on the frontend using React state.

🚀 Live Deployment:
Frontend deployed on Vercel:https://student-management-system-gray-nine.vercel.app/
🛠 Technologies Used
React.js
JavaScript
HTML
CSS
XLSX library (Excel download)
React Icons

💻 Run Locally:
Clone the repository:
git clone https://github.com/yourusername/student-management-system.git
Go to project folder:
cd frontend
Install dependencies:
npm install
Start the project:
npm start
The app will run at:
http://localhost:3000
